# Batch Copy X Bookmarks Links

Chrome 扩展：在 **https://x.com/i/bookmarks** 页面，从**侧边栏面板**采集收藏项链接，结果在面板内显示（地址之间换行），支持一键复制与采集范围控制。

## 安装（开发者模式）

1. 打开 Chrome，地址栏输入 `chrome://extensions/`
2. 打开右上角「开发者模式」
3. 点击「加载已解压的扩展程序」，选择本目录 `BatchCopyTwitterBookmarksLinks`
4. 访问 https://x.com/i/bookmarks ，等待收藏列表加载完成

## 使用

- **打开面板**：在 x.com/i/bookmarks 页面点击浏览器工具栏中的**扩展图标**，会打开本扩展的**侧边栏面板**（不再在页面右下角显示按钮）。
- 在面板中点击 **「采集收藏链接」**，会采集当前页面的收藏推文链接，结果在面板下方显示，每条地址一行。
- 使用 **「复制到剪贴板」** 可一次性复制全部链接（换行分隔）。

## 采集范围控制（面板内）

在面板的「采集范围」区域可控制：

- **仅采集当前视窗内可见的链接**：勾选后只采集当前滚动位置能看到的推文链接，不包含需要向下滚动才出现的。
- **去重**：勾选后对链接去重（默认勾选）。
- **最多采集 N 条**：填写数字后只保留前 N 条链接；填 **0** 表示不限制。

## 修改底层规则（选择器）

若需要改「哪些元素算收藏项」（例如换一种链接选择器），可编辑 `content.js` 中的常量：

- **SELECTOR**：当前为 `a[href*="/status/"]`，即所有包含 `/status/` 的链接。可改成更窄的选择器（例如只采某容器内的链接）。

改完后在 `chrome://extensions/` 中对该扩展点击「重新加载」即可生效。
